CommandHelper 다운로드

CommandHelper 를 통해 구동되는 MethodScript 인터프리터는 간편하게 명령어를 변경하고, 원하는 기능을 플러그인과 거의 비슷한 수준으로 추가할 수 있는 강력한 스크립팅 언어입니다. 간단하면서도 프로그래밍 언어의 기본적인 구조를 갖추고 있어 다른 프로그래밍 언어를 배울 때에도 큰 도움이 됩니다.

  • 최신 버전 다운로드
  • Backport(구 버전 Minecraft 용) 다운로드

확장 기능을 통한 기능 확장

확장 기능은 CommandHelper 의 확장성을 개선하여 Citizens, PlaceholderAPI 등의 플러그인, CHFiles 같은 확장기능을 통해 좀 더 많은 기능을 사용할 수 있습니다.
